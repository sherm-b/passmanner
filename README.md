
# passmanner

#### Disclaimer: This is a beginner student project and is *NOT SECURE*! Like at all! Use at your own risk.

---

A simple, CLI-based password manager in python.  
Allows users to add, list, search for, and delete account details stored in a .csv file with a nice workflow.  
Final project for CS162P at Lane Community College.
